# Piped CE Autoloaders 1.0.0

## Summary

This is the first stable release of Piped CE Autoloaders for RimWorld 1.6. It
provides three independently configured ammunition pipe networks backed by
Vanilla Expanded Framework and reloads adjacent turrets through Combat
Extended's native autoloader behavior.

## Highlights

- Three color-coded networks with exact CE ammo-set and round bindings.
- Powered pipe-backed autoloaders with native partial, shortage, and cancellation behavior.
- Configurable 0.1x-5.0x reload speed and 100-10,000-round magazine capacity.
- Normal and hidden pipe variants on the same network.
- Complete English, Simplified Chinese, French, German, Russian, and Spanish catalogs.

## Updating existing colonies

Rebinding a network changes existing stored and buffered counts to the newly
selected round after restart. Empty magazines before lowering capacity. Existing
1x2 storage from development builds expands to 2x2 in place, so inspect nearby
walls, roofs, paths, and pipe connections after updating.
