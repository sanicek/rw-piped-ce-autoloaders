# Piped CE Autoloaders 1.0.3

## Summary

This patch normalizes the physical behavior of ammunition-network buildings.
All player-facing machinery now provides the same low Combat Extended cover,
while magazines and autoloaders remain traversable at a substantial path cost.

## Changes

- Reduce ammunition magazines from full wall-height fill to 0.5 fill, producing
  0.88 m Combat Extended cover.
- Make ammunition magazines pass-through only with path cost 50.
- Add path cost 50 to piped autoloaders.
- Validate the shared magazine and autoloader cover and pathing contract during
  package validation.

## Update risk

Existing ammunition magazines adopt the new properties when their Defs load.
They no longer block low lines of fire like walls, and pawns can cross them when
the alternative route is sufficiently expensive. Their footprint, stored
resource, and hit points are unchanged.
