# Piped CE Autoloaders 1.0.1

## Summary

This patch makes Combat Extended ammo sets with identical caliber labels
distinguishable in Mod Settings. Distinct CE sets remain selectable and existing
saved selections retain their DefName identity.

## Changes

- Append the technical DefName only when multiple selectable ammo sets share one
  label.
- Show the same disambiguated label in the selected-value button and its menu.
- Remove migration guidance for development-only magazine footprints that were
  never released.
- Add the maintainer's AI-tool disclosure to the Workshop description.
