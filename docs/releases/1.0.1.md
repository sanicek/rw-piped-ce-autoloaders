# Piped CE Autoloaders 1.0.1

## Summary

This patch collapses redundant Combat Extended ammo-set options that represent
the same physical pipe resources. Genuinely distinct same-label sets remain
disambiguated, and existing saved selections retain their DefName identity.

## Changes

- Collapse same-label ammo sets whose physical rounds are identical to, or a
  subset of, a broader set.
- Append the technical DefName only when nonredundant same-label sets remain.
- Remove migration guidance for development-only magazine footprints that were
  never released.
- Add the maintainer's AI-tool disclosure to the Workshop description.
