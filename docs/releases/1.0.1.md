# Piped CE Autoloaders 1.0.1

## Summary

This patch groups Combat Extended ammunition by its physical item category
instead of exposing weapon-specific ammo sets. The mod derives the internal
containing set CE requires for the selected exact round. Existing saved
selections retain their DefName identity.

## Changes

- Present one caliber option per stable CE physical-ammunition category.
- Combine rounds from every weapon ammo set using that category and derive a
  deterministic containing set whenever the exact round changes.
- Remove migration guidance for development-only magazine footprints that were
  never released.
- Add the maintainer's AI-tool disclosure to the Workshop description.
